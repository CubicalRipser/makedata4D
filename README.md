## makedata4D

making a 4D voxel random blur data 

## Usage:

MD4D [options]

## options

### --help
 print help

### --nsize  n
 real data size, n^4
  
### --ssize  s 
 pattern size, once we make a random data of s^4, and spread it into s^4
  
### --radius r  
 when we spread the pattern into real data, r is the blur radius.

## output filename 
MD4D_n_s_r.complex
